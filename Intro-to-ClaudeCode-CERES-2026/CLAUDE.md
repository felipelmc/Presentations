# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository context

This directory is part of a broader `Presentations` repository maintained by Felipe Lamarca. Each subdirectory holds materials for a specific talk or workshop. This folder (`Intro-to-ClaudeCode-2026`) is for an introductory presentation on Claude Code.

Other presentations in the parent repo follow the same flat structure: slides, notes, and any demo code live together in the subdirectory with no build system.

## Presentation conventions

Slides are typically authored in a tool of the presenter's choice (e.g., Quarto, LaTeX Beamer, PowerPoint). Demo scripts and code snippets accompanying the talk live alongside the slide source in this directory.

## Language

The presenter writes primarily in Portuguese (pt-BR). Commit messages and file names in this repo are often in Portuguese.
